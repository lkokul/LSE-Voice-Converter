# LSE-Voice-Converter
A computer vision application using MediaPipe Gesture Recognition to translate Spanish Sign Language (SSL) into spoken audio in real-time via pyttsx3 and PyAudio.

## Real time Spanish Sign Language (LSE) - voice convertor
Nowadays are some sort of transactions —whether face-to-face or via phone call— require verbal communication. Since the other person doesn't always know sign language or it's not even available the option to do a video call, in this situations, a person who communicates using sign language needs an interpreter to translate into audio. This kind of situations are frustating and can create a barrier between the people on the call. This project is created to help in this type of situations, when an interpreter it's not available or you want to establish a direct, independent channel of communication without relying on third-party assistance.

## Installation
### Prerequisites
* **Python 3.11** (Recommended)
* **Conda** (Package management) [Install Conda here](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html)

### Setup
To clone the repository
```bash
git clone https://github.com/lkokul/LSE-Voice-Converter.git
```

To initiate the program, follow one of the next:
### Automated Setup (for Windows):
Run the init.bat file. It will handle the environment and dependencies by itself.

### Manual Setup
To install this project:
```bash
conda create -n lse-converter python=3.11
conda activate lse-converter
pip install -r requirements.txt
```
- Execute `main.py` file
